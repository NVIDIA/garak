# Garak calibration model bags

The calibration model-bag documentation now lives in the
[calibration reference](https://docs.garak.ai/en/latest/reporting.calibration.html).
This file remains as a stable destination for existing links.

The model lists and run configurations used for each calibration release are
archived as machine-readable CSV and YAML files under [`bags/`](bags/). The
release catalog and current-release pointer are in [`bags/index.yaml`](bags/index.yaml).
The generated calibration statistics remain in the versioned `calibration-*.json`
files in this directory.
